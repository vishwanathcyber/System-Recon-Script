# System-Recon-Script

## Overview

A beginner cyberscurity project written in Bash that automates collection of Linux system information.

## Planned Features

- OS Information Collection
- User Enumeration
- Network Information Collection
- Open Port Discovery
- Running process Enumeration
- Automated Report Generation
