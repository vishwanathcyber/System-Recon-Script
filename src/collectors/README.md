# Collectors

The collectors module is responsible for gathering system information from the Linux operating system.

## Planned Collector Modules

- CPU Information
- Memory Information
- Disk Information
- Network Information
- User Information
- Running Services
- Process Information
- Installed Packages
- Kernel Information

## Purpose

Each collector is responsible for collecting one specific type of system information. This modular design improves maintainability, testing, and scalability.

Future versions will separate each collector into its own Bash script.
